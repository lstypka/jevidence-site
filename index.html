Modern & intuitive!

Show results of tests executions in a modern & clear way

jEvidence allows you to present tests executions as a web page in a much clearer way than the existing solutions.

Key features:

  • Tests trend : Check how many tests have been added and how their status has been changed
  • Statistics : Check how execution duration and average test duration have been changed
  • Performance : Compare two executions and check time differences between them
  • Testing frameworks : jEvidence supports jUnit and testNG frameworks
  • Logging frameworks : jEvidence supports java.util.logging.Logger and log4j frameworks. Each log of these frameworks is saved as test step

Quick start

To get started with jEvidence, add the appropriate maven dependency, depending on your testing framework:

TestNg

<dependency>
    <groupId>pl.stypka.jevidence</groupId>
    <artifactId>jevidence-testng</artifactId>
    <version>1.2.1</version>
</dependency>

jUnit

<dependency>
    <groupId>pl.stypka.jevidence</groupId>
    <artifactId>jevidence-junit</artifactId>
    <version>1.2.1</version>
</dependency>

Next, register appropriate listener / runner to your tests

TestNg

@Listeners(JEvidenceTestNgListener.class)
public class AbstractTest {
}

jUnit

@RunWith(JEvidenceJUnitRunner.class)
public class AbstractTest {
}

Create jevidence.properties file with the following content

jevidence.reportPath=C:\\reports\jevidence
jevidence.removeExecutions=true
jevidence.maxNumberOfExecutions=50
jevidence.embedded=true

At this point you can register jEvidence to your logger.

Note, listener / appender should be registered once only. Otherwise, the same test step may be duplicated

java.util.logging.Logger

<dependency>
    <groupId>pl.stypka.jevidence</groupId>
    <artifactId>jevidence-java-logger</artifactId>
    <version>1.2.1</version>
</dependency>
java.util.logging.Logger LOG = java.util.logging.Logger.getLogger(AbstractTest.class.getName());
LOG.getParent().addHandler(new JEvidenceUtilLoggerHandler());

Log4j

<dependency>
    <groupId>pl.stypka.jevidence</groupId>
    <artifactId>jevidence-log4j</artifactId>
    <version>1.2.1</version>
</dependency>
org.apache.log4j.Logger LOG4J = org.apache.log4j.Logger.getLogger(AbstractTest.class.getName());
LOG4J.getParent().addAppender(new JEvidenceLog4jHandler());
